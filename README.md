# Configuração de Roteador Sem Fio e Cliente - Cisco Packet Tracer

## Sobre o Projeto

Este laboratório foi desenvolvido no Cisco Packet Tracer com o objetivo de aprender os conceitos básicos de configuração de uma rede doméstica sem fio.

Durante a atividade, foi realizada a configuração de um roteador wireless, permitindo que dispositivos da rede se conectassem à Internet de forma segura através de uma rede Wi-Fi configurada.

## Topologia da Rede

![Topologia da Rede](topologia.png)

A topologia é composta pelos seguintes dispositivos:

- Home Wireless Router
- Cable Modem
- Splitter
- Internet (Cloud)
- Office PC
- Bedroom PC
- TV
- Servidor Skills For All

## Atividades Realizadas

- Configuração inicial do roteador sem fio
- Configuração do nome da rede (SSID)
- Configuração da senha de acesso ao Wi-Fi
- Conexão de dispositivos à rede wireless
- Verificação da conectividade da rede
- Teste de acesso à Internet

## Tecnologias Utilizadas

- Cisco Packet Tracer
- Redes Wireless (Wi-Fi)
- TCP/IP
- Roteador Doméstico

## Aprendizados

Com este laboratório, aprendi:

- Como configurar um roteador sem fio
- Como alterar o nome da rede (SSID)
- Como definir uma senha para proteger o Wi-Fi
- Como conectar dispositivos a uma rede wireless
- Como funciona uma topologia básica de rede doméstica
- A importância da segurança em redes sem fio
