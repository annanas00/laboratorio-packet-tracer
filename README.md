# Laboratório de Redes Cisco

Repositório com atividades práticas, laboratórios e simulações desenvolvidos durante meus estudos em Redes de Computadores por meio dos cursos da Cisco Networking Academy.

## Conteúdos Estudados

- Conceitos básicos de redes
- Endereçamento IPv4 e IPv6
- Configuração de roteadores e switches
- DHCP
- DNS
- Redes sem fio (Wireless)
- Testes de conectividade
- Serviços de rede
- Simulações no Cisco Packet Tracer

## Ferramentas Utilizadas

- Cisco Packet Tracer
- Cisco Networking Academy

## Objetivo

Registrar minha evolução prática na área de Redes de Computadores e consolidar os conhecimentos adquiridos durante a formação.
